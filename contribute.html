02 How to contribute
  1. Design your icon — trace an image above, or edit the JSCad code and click ▶ Run to preview. main() must return a single 2D shape.
  2. Submit a PR — click Contribute to Gallery…, give it a name, keywords, and a filename id (e.g. hex-wrench), then Open on GitHub →. GitHub pre-fills a single SVG file under images/custom/ and walks you through forking & opening a pull request — no CLI or authentication needed. Once merged, it appears for everyone under Icon → Gallery.
  3. Use it now — while your PR is in review, click Export SVG ↓ to download your icon as a vector file. Then on the main app go to Icon → Upload SVG… to use it immediately in your current session.

See CONTRIBUTING.md for file format details and local development instructions.

Contribute to Gallery as PR

Name — shown in the icon picker.

Keywords — comma-separated terms people can search by.

Filename id — lowercase letters, digits, dashes. It becomes images/custom/<id>.svg.

Clicking Open on GitHub opens a new tab on github.com with the SVG file pre-filled. GitHub will offer to fork the repo and open a pull request — no authentication required here. Once merged it appears under Icon → Gallery for everyone.
