02 How to contribute
  1. Design your icon — edit the JSCad code above and click ▶ Run to preview it. Pick a standard from the left to load and modify an existing definition, or choose New standard to start from the template.
  2. Submit a PR — click Submit / Modify Standard…, enter a standard ID (e.g. din912), then Open on GitHub →. GitHub pre-fills the file and walks you through forking & opening a pull request — no CLI or authentication needed.
  3. Use it now — while your PR is in review, click Export SVG ↓ to download your icon as a vector file. Then on the main app go to Icon → Custom → Upload SVG… to use it immediately in your current session.

See CONTRIBUTING.md for file format details and local development instructions.

Submit New Standard as PR

Choose an ID for this standard (lowercase letters, digits, dashes). It becomes the filename standards-jscad/<id>.js.

Clicking Open on GitHub opens a new tab on github.com with the file pre-filled. GitHub will offer to fork the repo and open a pull request — no authentication required here.

The modified file will be copied to your clipboard and GitHub's editor will open. Select all (Ctrl+A) and paste (Ctrl+V) to replace the file, then submit a pull request.
